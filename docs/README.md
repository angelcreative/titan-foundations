# Titan Docs Index

This folder is organized by documentation intent:

- `core/`: global rules, token architecture, terminology, and quality guidelines.
- `components/`: component behavior/spec docs tied to `component-specs/*.json`.
- `anatomies/`: reusable UI anatomy patterns for LLM-driven generation.
- `integration/`: ownership, decision policy, migration, and audit runbooks.
