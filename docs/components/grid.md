# Grid

Content below the navbar uses a **16-column grid**, centered, with max width 1920px (responsive 1440px / 1280px).

## Key tokens

| Token | Purpose |
|------|---------|
| `--layout-grid-gap` | Gap between columns and rows |
| `--layout-grid-max-width` | Max container width (1920px) |

## Rules

- Use the 16-column structure consistently.
- Keep vertical spacing between stacked sections.
- For responsive behavior, respect template breakpoints.

Spec: `component-specs/grid.json`.
