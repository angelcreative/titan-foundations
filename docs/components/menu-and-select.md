# Menu and Select

Operational contract for Menu and Select patterns with Titan tokens and React Aria structure.

## Menu tokens

| Token | Purpose |
|------|---------|
| `--menu-min-width` | Minimum width |
| `--menu-max-width` | Maximum width |
| `--menu-bg` | Menu background |

## Select tokens

| Token | Purpose |
|------|---------|
| `--select-item-radius` | List item radius |

## Rules

- Keep Menu and Select structure aligned with React Aria primitives.
- Use token-driven states for focus/pressed/selected.
- Keep destructive actions clearly differentiated.

Specs: `component-specs/menu.json`, `component-specs/select.json`.
